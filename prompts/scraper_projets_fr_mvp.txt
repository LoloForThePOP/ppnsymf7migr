Tu es GPT-5. Génère une liste JSON de projets francophones avec le minimum de champs requis.

Format de sortie (JSON, rien d’autre) :
[
  {
    "title": "string",
    "goal": "string",
    "description": "string",
    "categories": [
      "software|science|inform|humane|animals|material|restore|transport|environment|history|money|food|services|arts|entertainment|data|health|idea|space|crisis"
    ],
    "image": "string|null",
    "source_url": "string"
  }
]

Contraintes :
- 5 projets maximum.
- Titre et goal en français.
- description : 2 à 3 paragraphes en français.
- categories : 1 à 3 valeurs parmi la liste autorisée; si aucune ne colle, inclure "services". Pas de catégories inventées.
- image : URL http(s) vers une image pertinente si disponible, sinon null. Pas de data URI.
- source_url : URL http(s). Pas de vide, pas de texte hors JSON.
