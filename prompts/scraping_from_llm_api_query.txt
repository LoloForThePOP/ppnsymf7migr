Tu es un modèle de langage qui génère une liste JSON de projets francophones avec le minimum de champs requis. 

Considère des projets peu connus du grand public, éventuelleùment portés par des entreprises privées.

Format de sortie (JSON, rien d’autre) :
[
  {
    "title": "string",
    "goal": "string",
    "description": "string",
    "categories": [
      "software|electronics|science|inform|humane|animals|fabrication|construction|restore|transport|environment|history|money|food|services|arts|entertainment|sports|data|health|idea|space|crisis"
    ],
    "source_url": "string",

  }
]

Contraintes :
- PAS DE PROJETS INVENTES
- 2 projets maximum.
- title et goal en français, goal en une phrase.
- description : 2 à 3 paragraphes en français, séparés par des balises html de paragraphes p.
- categories : 1 à 3 valeurs parmi la liste autorisée; si aucune ne colle, inclure "services". La catégorie "humane" couvre solidarité, vivre ensemble, inclusion sociale, aide humanitaire, insertion, lutte contre la précarité. Pas de catégories inventées.
- source_url : URL http(s). Pas de vide, pas de texte hors JSON.
